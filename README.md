## Project Description
A JavaFX desktop application that allows customers to create accounts, search for movies, view showtimes, purchase tickets, and manage reservations. Managers can generate reports and analyze sales data.

## Technology Stack
- Java
- JavaFX
- GitHub
- Jira
- Text File Storage

## Sprint 1 User Stories
- US-1 Customer Registration
- US-2 Customer Login
- US-3 Search Movies
- US-4 View Showtimes
